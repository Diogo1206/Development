Conjunto é uma coleção bem definida de objetos, chamados Elementos
Ex: B = {m,t,c,v}    N = {11,2,3,4, ...}

C está constituído por elementos "x" tal que a altura de "x" ´pe maior de 1,75 m
C = {x | altura de x > 1,75 m}

Propriedade que caracteriza os elemento de C:
P(x) : a altura de x é maior que 1,75m
C = {x | P(x)}

C - Conjunto
= - está constituído
x - pelos elementos X
| = tal que
P(x) - verifica P(x)

D = conjunto de números naturais maiores ou iguais a 5.

Representação explícita: D = {5, 6, 7, 8, ...}
Representação implícita: D = {x | x £ N e x >= 5}

O conjunto vazio Ø(alt+0216), é o conjunto que não tem elementos.
Ø = {x | x £ N, x > 5 e x < 0}
Ø = {x | x £ Z, 2x-1=0}

Os conjuntos A e B são iguais quando tem os mesmos elementos.
Notação: A = B
Ex: A = { 1, 3, a} B = {3, a, 1} C = {1, 3, 1, a} D = {2, 3, a}
Os conjuntos A, B e C são iguais, A=B=C
A é diferente de D.

Um conjunto A está contido em um conjunto B se todo elemento de A é elemento de B
